# Mayerfeld Consulting – Final Assignment Project One

A browser-based calculator application developed as part of a **team collaboration project**.

## Live Demo
[Live Demo]((https://mayerfeld-consulting-final-assignme.vercel.app/)) 

## Project Overview
This project is a calculator web application focused on implementing core arithmetic operations and handling user input in real time. The goal was to build a functional and responsive calculator using fundamental frontend technologies.

## Key Features
- basic arithmetic operations (addition, subtraction, multiplication, division)
- real-time input handling
- dynamic display updates
- responsive layout
- clean and intuitive user interface

## Tech Stack
- HTML
- CSS
- JavaScript

## Collaboration
This project was developed as a **team collaboration**, simulating a real development workflow:
- working in a shared codebase (Git)
- task distribution and coordination
- collaborative problem solving
- code reviews and improvements

The experience reflects real-world teamwork in building interactive web applications.

This project demonstrates the team’s ability to:
